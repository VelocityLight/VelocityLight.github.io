MacOS install hexo

IT

关于Mac上工程迁移.

安装node.js

建议官网上下载: https://nodejs.org/en/
安装完成查看版本信息:
node -v; npm -v

安装hexo

sudo npm install hexo-cli -g
安装完成查看版本信息: hexo -v

初始化hexo工程

若无工程: hexo init
若有工程，工程下执行: npm install
